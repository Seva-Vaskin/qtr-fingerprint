# qtr-fingerprint
## Build
1. git clone https://github.com/quantori/qtr-fingerprint.git
2. cd qtr-fingerprint
3. git submodule update --init --recursive
4. Open qtr-fingerprint in VS Code:
   - configure project
   - build project
## Run
1. cd qtr-fingerprint/dist/lib/windows-x86_64
2. app.exe
